# Loose — AI Agent Launchpad on Robinhood Chain

Deploy, manage, and monetize autonomous AI agents on Robinhood Chain.

## Features

- **AI Agent Deployment** — Launch autonomous trading agents with pre-built strategies (Arbitrage, Rebalancer, Whale Tracker, Sniper)
- **Dashboard** — Real-time monitoring, pause/resume/stop controls, and performance analytics
- **Wallet Integration** — Seamless wallet connection via Privy on Robinhood Chain (Arbitrum L2)
- **$LOOSE Token** — Native utility token for the platform
- **Airdrop** — Earn points through tasks and referrals
- **Staking** — Lock $LOOSE tokens for rewards

## Tech Stack

- **Framework**: Next.js 16 (App Router, Turbopack)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Wallet**: Privy (@privy-io/react-auth)
- **Chain**: Robinhood Chain — Arbitrum-stack Ethereum L2 (Chain ID: 4663)
- **Database**: Supabase
- **AI**: OpenRouter

## Getting Started

```bash
# Install dependencies
npm install

# Copy environment variables
cp .env.example .env.local

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment Variables

See `.env.example` for required environment variables.
